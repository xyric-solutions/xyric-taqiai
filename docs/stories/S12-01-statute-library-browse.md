# S12-01 — Statute Library Browse (6 Codes + Sections)

## User Story

As a lawyer, I want to browse a built-in statute library of Pakistan's major codes and their sections so that I can find any section I need without flipping through bare-act books.

## Acceptance Criteria

- [x] Library includes 6 codes: PPC 1860, CrPC 1898, CPC 1908, MFLO 1961, SRA 1877, FCA 1964
- [x] Each code lists its sections
- [x] Lawyer can browse a code and select a section
- [x] Page is usable on mobile screen (375px+)

## Technical Notes

- Route / page: `/statute-search`
- Built-in statute library: sections for the 6 major codes
- Skill: LEGAL-08 (Statute & Citation Search)

## Definition of Done

- [x] All 6 codes browsable with their sections
- [x] Selecting a section opens its detail / search
- [x] Mobile view readable
- [x] Abdullah sign-off
