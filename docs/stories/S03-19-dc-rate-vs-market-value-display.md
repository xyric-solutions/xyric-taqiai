# S03-19 — DC Rate vs Market Value Side-by-Side Display

## User Story

As a lawyer, I want to see both DC rate value and market value side by side so that I understand which value is being used for which tax.

## Acceptance Criteria

- [x] Results show DC rate value and market value in a side-by-side comparison
- [x] Each tax line clearly labelled with which value it is based on (DC or Market)
- [x] Lawyer understands at a glance why each tax is different

## Technical Notes

- Live UI route: `/property-transfer/tax-calculator`
- Two-column result display (DC Rate vs Market Value)
- Helps lawyers explain FBR methodology to clients

## Definition of Done

- [x] Side-by-side view rendered clearly in UI
- [x] Each tax line shows which value was used
- [x] Reviewed by a practicing lawyer for clarity
- [x] Abdullah sign-off
