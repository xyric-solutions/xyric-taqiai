# S03-08 — Tax Summary Disclaimer

## User Story

As a lawyer, I want the tax summary PDF to include a disclaimer so that my client understands this is an estimate and must be verified.

## Acceptance Criteria

- [x] Mandatory disclaimer shown on result page: this is an estimate, verify with a qualified tax practitioner before submission
- [x] Same disclaimer present in the printable summary
- [x] Disclaimer not removable by lawyer — always included

## Technical Notes

- Disclaimer text is fixed — not user-editable
- Must appear on both the on-screen result and in the printable summary

## Definition of Done

- [x] Disclaimer shown on calculator result page
- [x] Disclaimer present in the printable summary
- [x] Cannot be hidden or removed
- [x] Abdullah sign-off
