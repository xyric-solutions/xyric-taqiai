# S08-06 — Print / Export Diary as PDF

## User Story

As a lawyer, I want to print or export my diary as a PDF so that I have a physical copy to take to court or share with my junior.

## Acceptance Criteria

- [ ] "Print / Export PDF" button available on diary page
- [ ] PDF output shows all 7 columns: Case Number | Last Date | Title | Court Name | Stage | Proceeding | Next Date
- [ ] PDF includes lawyer's name and export date at top
- [ ] PDF includes footer disclaimer: "Generated by TaqiAI — For personal use only"
- [ ] Currently active filter/sort is reflected in the PDF (if lawyer filtered by one court, PDF shows only that court's cases)
- [ ] PDF is A4 landscape format (fits 7 columns cleanly)

## Technical Notes

- Use browser print API (`window.print()`) with print-specific CSS stylesheet — simplest approach for v1
- Alternatively: server-side PDF generation (Puppeteer / react-pdf) if print CSS insufficient
- Landscape A4: `@page { size: A4 landscape; }`
- Hide navigation/sidebar in print view

## Definition of Done

- [ ] PDF export produces clean A4 landscape table with all 7 columns
- [ ] Lawyer name and date shown at top of PDF
- [ ] Active filter reflected in export
- [ ] Abdullah sign-off
