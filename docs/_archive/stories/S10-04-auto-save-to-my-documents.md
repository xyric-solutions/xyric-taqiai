# S10-04 — Auto-Save to My Documents

## User Story

As a lawyer, I want the generated document to auto-save to My Documents so that I don't lose my work and can find it later.

## Acceptance Criteria

- [x] Generated document automatically saves to "My Documents"
- [x] No manual save step required
- [x] Saved document is retrievable from the document library
- [x] Saved document is scoped to the logged-in lawyer

## Technical Notes

- Auto-save target: My Documents library
- Documents scoped per `userId`
- Skill spec: LEGAL-10 (Document OCR)

## Definition of Done

- [x] Auto-save to My Documents confirmed
- [x] Saved document retrievable later
- [x] Abdullah sign-off
