# S12-05 — Bookmark Frequently-Used Sections

## User Story

As a lawyer, I want to bookmark frequently-used sections so that I can reach the sections I rely on most without browsing every time.

## Acceptance Criteria

- [x] Lawyer can bookmark a section
- [x] Bookmarked sections are saved and recalled per user
- [x] Lawyer can remove a bookmark
- [x] Bookmarks are accessible for quick navigation back to a section

## Technical Notes

- Per-user bookmarks for sections in the statute library
- Route / page: `/statute-search`

## Definition of Done

- [x] Bookmark add / remove working
- [x] Bookmarks persist per user
- [x] Quick access to bookmarked sections
- [x] Abdullah sign-off
